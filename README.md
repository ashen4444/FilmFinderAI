# FilmFinder AI

FilmFinder AI is an intelligent movie discovery and recommendation system that helps users find movies using natural language descriptions or by entering a movie title.

The system combines semantic search and recommendation techniques to identify movies based on meaning rather than simple keyword matching.

## Key Features

**1. Movie Recommendation**

* Users can enter a movie title.
* The system recommends similar movies based on semantic similarity.

**2. Memory-Based Movie Search**

* Users can describe a movie they remember.
* The system finds the most likely matching movies from the dataset.

Example queries:

* *"A space movie where astronauts travel through a wormhole"*
* *"A prison escape movie with a very intelligent main character"*

## Technologies Used

* Python
* Pandas
* NumPy
* Sentence Transformers
* Scikit-learn

## Dataset

The project uses the TMDb movie metadata dataset containing movie titles, genres, release dates, ratings, and plot overviews.

## Project Goal

The goal of FilmFinder AI is to demonstrate how semantic embeddings and vector similarity can be used to build an intelligent search system that understands the meaning of user queries.

This project is being developed as part of an AI university project focused on semantic search and recommendation systems.
